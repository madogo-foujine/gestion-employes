# Fonctionnalités détaillées

## 1. Employés
- Fiche complète : identité, contact, administratif, salaire.
- Photo de l'employé (affichage circulaire, reprise dans les documents).
- Recherche et filtres (par poste, par type de contrat).
- Documents joints par employé (contrat, CIN, diplômes…).

## 2. Paie (Maroc)
- **CNSS** : 4,48 % plafonné à 6000 DH (max 268,80 DH).
- **AMO** : 2,26 %.
- **Frais professionnels** : 35 % / 25 % plafonnés.
- **IR** : barème mensuel progressif, **modifiable** dans les paramètres.
- Déductions : absences, autres retenues, **avances** sur salaire.
- **Congés** : solde acquis / pris / restant.

## 3. Pointage
- Grille mensuelle (Présent / Absent / Congé / Repos).
- Calendrier annuel récapitulatif.
- Les absences sont reportées automatiquement sur la paie.

## 4. Documents générés (PDF / HTML / Excel)
- Bulletin de paie (PDF et HTML).
- Attestation de travail / de salaire.
- Contrat de travail (CDI / CDD).
- État de paie mensuel (HTML et Excel).
- Déclaration CNSS.
- Logo et signature/cachet automatiques ; **numérotation séquentielle** + registre.

## 5. Tableau de bord & analyses
- Indicateurs : effectif, masse salariale brute/nette, net moyen, total IR.
- Graphique des salaires, évolution mensuelle de la masse salariale.
- Simulateur de salaire, alertes RH (anniversaires, fins de contrat).

## 6. Données & sécurité
- Stockage dans un fichier Excel + JSON de configuration.
- Sauvegardes automatiques avant chaque enregistrement.
- Mot de passe et rôles (administrateur / comptable).
- Import / export CSV.

## 7. Interface
- Thème clair / sombre.
- Barre de menus + barre d'outils.
